# Fashion-MNIST Classification using MLP

## Problem Description
This project implements a Multilayer Perceptron (MLP) to classify clothing images from the Fashion-MNIST dataset.

## Dataset
https://github.com/zalandoresearch/fashion-mnist

## Model Architecture
- Input Layer: 784 neurons
- Hidden Layer 1: 128 neurons
- Hidden Layer 2: 64 neurons
- Output Layer: 10 neurons

## Regularization
- Dropout
- Batch Normalization

## Experiments
1. ReLU Activation
2. Sigmoid Activation

## Results
| Experiment | Accuracy |
|-----------|----------|
| ReLU | 88.47% |
| Sigmoid | 87.17% |
